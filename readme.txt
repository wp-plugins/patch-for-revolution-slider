This plugin repairs the arbitrary file download vulnerability in Revolution Slider plugin.

== Description ==

This plugin repairs the arbitrary file download vulnerability in Revolution Slider plugin. This vulnerability is critical because anyone can download your wp-config.php or any other file from the server without even logging in.

== Installation ==

Just upload the plugin to wp-content/plugins/, activate it and you don't need to worry anymore about anyone trying to access your private information.

== Frequently Asked Questions ==

= Why should I use this plugin? =
It's useful to use it if your Revolution Slider is in a template and you can't update it at the moment (eg. if the template has been modified).
